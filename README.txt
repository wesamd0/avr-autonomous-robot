- Dans le répertoire «lib», nous avons tous les classes nécessaire pour les deux robots. La classe «robot1» et «robot2» incluent certains
  autres classes et les classes «distance» et «lignes» sont responsables de parcours des robots respectivement. Pour le robot 2, le calcul
  de distance est inclue dans la classe «lignes» et le même chose s'applique pour le robot 1 pour déterminer sa position.

- Dans le répértoire app1, le fichier main accompagné de son Makefile sont responsables de l'éxcution du code qui est responsable de
  centrement de robot et des modes définis après avoir réussis à se centrer.

- Dans le répértoire app2, le fichier main accompagné de son Makefile sont responsables de l'éxcution du code qui est responsable de
  d'effectuer le parcours et de calculer les distances et de passer aux autres modes définis.
